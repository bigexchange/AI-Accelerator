Turn an email into a calendar event

Someone sends you scheduling details buried in a thread. Instead of copying times and links manually, point Cowork at the email and it creates the event with everything filled in.

📅
What you'll have
A fully populated Google Calendar event — title, time, attendees, video link, and description — created from an email in seconds. No copy-pasting.

What you'll need

🖥️Claude Desktop
📧Gmail connector
📅Google Calendar
😩 Without this

Read email → switch to Calendar → create event → paste title → find time → copy Zoom link → paste → add attendees one by one. 3+ minutes of switching tabs.

✨ With Cowork

One prompt → Cowork reads the email, extracts every detail, and creates the event live in your Calendar. Under 30 seconds.

1

Point Cowork at the email

Describe the email by sender, subject, or topic. Cowork finds it, reads it, and creates the event.

Paste into Cowork Find the most recent email from [sender name or email] about [meeting topic / subject line]. Read it and create a Google Calendar event with all the details — title, date, time, video link or location, and attendees. Add the email body as the event description so I have context in the invite.

3

For back-and-forth scheduling threads

Sometimes the confirmed time is buried across a whole email chain. Cowork can read the full thread and pull the final agreed time.

For email threads Find the email thread with [name] about [topic]. Read the full thread and figure out the time we agreed on. Create the calendar event with the confirmed time, all attendees, and any video link mentioned. Use the thread subject as the event title and add a brief summary as the description.
4

Batch mode — catch up on a backlog

Have a pile of scheduling emails you haven't calendared yet? Run this to catch up all at once.

Batch version Search my Gmail from the last 5 days for any emails that contain meeting requests, scheduling confirmations, or event invites I haven't added to my calendar yet. For each one you find, create the Google Calendar event with full details. List what you created at the end so I can review.
⚠️
Quick review recommendedFor batch mode, always scan your calendar after — Cowork might pick up automated confirmations or reminders that don't need a new event. A 10-second check saves confusion.

Pro tips

🔄
Combine with meeting prepAfter the event is created, immediately run the meeting prep guide on it. One prompt to create the event, one to get your brief — fully set up before you even reply to the email.
📧
Draft your reply tooAdd "...and draft a reply confirming attendance and asking if there's anything I should bring." It'll land in your Gmail drafts folder, ready to send.
🕐
Time zones are handledIf the email mentions a time zone (e.g., "2pm PST"), Cowork converts it to your local time when creating the event. Double-check for cross-timezone meetings just in case.
